🚀 立即开始（3步走）
================================================================================

第1步：启动后端
--------------
双击运行: start_backend.bat

或者手动运行:
  cd backend
  pip install -r requirements.txt
  python -m app.main

验证: 浏览器访问 http://localhost:8000/health


第2步：启动前端
--------------
双击运行: start_frontend.bat

或者手动运行:
  cd frontend
  npm install
  npm run dev

验证: 浏览器访问 http://localhost:5173


第3步：测试功能
--------------
1. 在聊天界面输入："你好"
2. 点击发送按钮
3. 应该收到AI的回复
