# Hubu Agent Frontend

## 简介
简化的智能对话系统前端，基于 Vue 3 和 Element Plus 构建。

## 目录结构
```
src/
├── apis/            # API调用封装
├── components/      # 可复用组件
├── pages/          # 页面组件
├── router/         # 路由配置
├── App.vue         # 根组件
└── main.ts         # 入口文件
```

## 启动方式
```bash
npm install
npm run dev
```

访问 http://localhost:5173 查看应用
