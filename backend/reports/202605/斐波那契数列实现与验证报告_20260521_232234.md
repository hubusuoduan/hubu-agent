# 斐波那契数列实现与验证报告

## ✅ 函数定义

```python
def fib(n):
    """返回第 n 项斐波那契数（F(0)=0, F(1)=1）"""
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```

- **算法**：迭代法，时间复杂度 O(n)，空间复杂度 O(1)
- **健壮性**：天然支持 `n = 0`；对非负整数输入零异常风险
- **使用示例**：`fib(0) → 0`，`fib(10) → 55`
